## 3. Application Entry Point

**File:** `linphone-desktop/Linphone/main.cpp`

Flow:
1. Windows STA COM init (`CoInitializeEx` APARTMENTTHREADED)
2. Register AUMID + COM server for Windows toast notifications (`DesktopNotificationManagerCompat`)
3. `App::create(argc, argv)` ‚Äî singleton `QApplication` subclass
4. If secondary instance ‚Üí `app->sendCommand()` then exit (single-instance guard)
5. If primary ‚Üí `app->initCore()` ‚Üí `app->exec()` (Qt event loop)
6. On `App::StatusCode::gRestartCode` ‚Üí loop restarts (handles in-app language change restart)

## 4. Application Class ‚Äî `App` (core layer)

**Files:** `Linphone/core/App.hpp` / `App.cpp`  
Extends: `SingleApplication` (cross-platform single-instance), `AbstractObject`

### Key responsibilities
- Creates `CoreModel` on a dedicated `Thread` (Linphone thread, separate from Qt main thread)
- Registers all C++ types with QML engine (`qmlRegisterType`, `qmlRegisterSingletonType`)
- Manages QML engine + `MainWindow.qml` root
- Owns: `Notifier`, `EventCountNotifier`, system tray icon
- Exposes to QML: `accounts`, `currentAccount`, `calls`, `chats`, app version, SDK version

### Thread model
- **Qt main thread** ‚Äî UI, QML, `App`, all `*Core` / `*Gui` objects
- **Linphone thread** ‚Äî `CoreModel`, all `*Model` objects, `linphone::Core` iterate loop (20 ms timer)
- Cross-thread calls use `App::postModelAsync` / `App::postModelSync` (QMetaObject::invokeMethod)

## 5. Architecture Layers

The app has a strict 3-layer architecture per domain:

```
QML (view)  ‚Üê‚Üí  *Gui / *Core (Qt main thread)  ‚Üê‚Üí  *Model (Linphone thread)  ‚Üê‚Üí  linphone::Xxx (SDK)
```

| Layer | Naming pattern | Thread | Role |
|---|---|---|---|
| View | `*.qml` | Qt main | Declarative UI, binds to Gui/Core via Q_PROPERTY |
| Core/Gui | `XxxCore`, `XxxGui`, `XxxList`, `XxxProxy` | Qt main | Qt wrappers; proxy/filter lists for QML; emit signals |
| Model | `XxxModel` | Linphone thread | Owns SDK objects; calls `linphone::Xxx` API |
| SDK | `linphone::Xxx` | Linphone thread | liblinphone C++ API |
