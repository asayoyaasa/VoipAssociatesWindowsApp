## Critical gotchas — must know

1. **VS path is `\18\` not `\2022\`** — always use `\18\`
2. **xerces-c option is `BUILD_XERCESC_SHARED_LIBS`** (has a C) — `BUILD_XERCES_SHARED_LIBS` is wrong
3. **srtp CMakeLists.txt is patched** — do not `git reset` that file
4. **Do NOT init `linphone-desktop/external/linphone-sdk` submodule** — points to BC's GitLab (offline), bypassed via `LINPHONE_QT_ONLY=ON`
5. **Do NOT init `external/google/crashpad`** — bypassed by `LINPHONE_QT_ONLY=ON`
6. **gitlab.linphone.org has been unreachable** — use GitHub mirrors for any missing packages
7. **`C:\voipapp` junction must exist** — Qt6LinguistTools cmake macros break with spaces in the source path. If junction is missing: `mklink /J C:\voipapp "C:\Users\tigaa\OneDrive\Documents\VOIP ASSOCIATES WINDOWS APP"`
8. **`jsoncpp` not `jsoncpp_lib`** — the installed lib is `jsoncpp.lib`; the original targets file had the wrong name
9. **`GENERATE_HTML = NO` in Doxyfile.in** — required to prevent Windows MAX_PATH overflow during genwrapper doxygen run; do not revert to YES
10. **Qt6NetworkAuth is not installed** — `ENABLE_APP_OAUTH2` must remain OFF (default); OIDCModel is stubbed out
